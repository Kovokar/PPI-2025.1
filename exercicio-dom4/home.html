Demonstração de APIs

YouTube Data API v3

Ver Projeto
API para acessar dados do YouTube como vídeos, canais e estatísticas.
Autenticação: Requer API Key do Google Cloud Console
curl -X GET "https://www.googleapis.com/youtube/v3/search?part=snippet&q=javascript&type=video&key=YOUR_API_KEY"
Substitua YOUR_API_KEY pela sua chave da API

Spotify Web API

Ver Projeto
API para acessar dados musicais do Spotify: músicas, álbuns, artistas e playlists.
Autenticação: OAuth 2.0 com Client ID e Client Secret
curl -X GET "https://api.spotify.com/v1/search?q=coldplay&type=artist" \ -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
Primeiro obtenha o token de acesso com suas credenciais

Google Maps Platform API

Ver Projeto
APIs para mapas, geocodificação, roteamento e dados de localização.
Autenticação: Requer API Key do Google Cloud Console
curl -X GET "https://maps.googleapis.com/maps/api/geocode/json?address=São+Paulo&key=YOUR_API_KEY"
Configure restrições de domínio para segurança

PokéAPI

Ver Projeto
API gratuita com dados completos sobre Pokémon, habilidades e tipos.
Autenticação: Não requer - API totalmente gratuita!
curl -X GET "https://pokeapi.co/api/v2/pokemon/pikachu"
Sem necessidade de chaves de API ou tokens
